# Configuration File

The CLI reads configuration from `~/.ee/config.yaml`.

## File Location

```
~/.ee/config.yaml
```

## Configuration Schema

```yaml
envoy:
  version: 1.37.0
  log_level: info

proxy:
  port: 10000
  admin_port: 9901

marketplace:
  url: https://envoy-ecosystem.io

cache:
  dir: ~/.ee/cache
  ttl: 24h

editor: vim
```

## Configuration Sections

### Envoy Settings

```yaml
envoy:
  version: 1.37.0      # Default Envoy version
  log_level: info       # Default log level
```

### Proxy Settings

```yaml
proxy:
  port: 10000          # Default proxy port
  admin_port: 9901     # Default admin port
```

### Marketplace Settings

```yaml
marketplace:
  url: https://envoy-ecosystem.io  # Marketplace URL
```

### Cache Settings

```yaml
cache:
  dir: ~/.ee/cache     # Cache directory
  ttl: 24h             # Cache TTL
```

### Editor

```yaml
editor: vim            # Default editor for editing manifests
```

## Managing Configuration

You can also manage configuration using the [ee config](/docs/cli/config) command:

```bash
# Set a value
ee config set envoy.version 1.37.0

# Get a value
ee config get proxy.port

# List all configuration
ee config list

# Reset to defaults
ee config reset
```
