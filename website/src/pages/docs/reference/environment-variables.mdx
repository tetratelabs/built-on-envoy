# Environment Variables

Configure CLI behavior with environment variables:

## Available Variables

### `BOE_ENVOY_VERSION`
Default Envoy version

```bash
export BOE_ENVOY_VERSION=1.36.0
```

### `BOE_PROXY_PORT`
Default proxy port

```bash
export BOE_PROXY_PORT=8080
```

### `BOE_ADMIN_PORT`
Default admin port

```bash
export BOE_ADMIN_PORT=9000
```

### `BOE_LOG_LEVEL`
Default log level

```bash
export BOE_LOG_LEVEL=debug
```

### `BOE_MARKETPLACE_URL`
Custom marketplace URL

```bash
export BOE_MARKETPLACE_URL=https://custom-marketplace.example.com
```

### `BOE_CACHE_DIR`
Cache directory location

```bash
export BOE_CACHE_DIR=~/.cache/built-on-envoy
```

### `BOE_NO_COLOR`
Disable colored output

```bash
export BOE_NO_COLOR=1
```

## Example Usage

```bash
# Set environment variables
export BOE_ENVOY_VERSION=1.36.0
export BOE_LOG_LEVEL=debug

# Run CLI with these settings
<CLIName/> run --plugin rate-limiter
```
