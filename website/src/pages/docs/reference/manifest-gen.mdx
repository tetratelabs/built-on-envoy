## Field Reference


### `author`

Author or organization name

| | |
|---|---|
| **Type** | `string` |
| **Required** | Yes |
| **Min length** | 1 |


### `categories`

Functional categories of the extension

| | |
|---|---|
| **Type** | `array of string` |
| **Required** | Yes |
| **Allowed values** | `AI`, `Network`, `Observability`, `Performance`, `Transform`, `Security` |


### `description`

Short single-line description of the extension

| | |
|---|---|
| **Type** | `string` |
| **Required** | Yes |
| **Min length** | 1 |
| **Max length** | 200 |


### `examples`

Usage examples for the extension

| | |
|---|---|
| **Type** | `array of object` |
| **Required** | Yes |
| **Min items** | 1 |

#### Properties

##### `code`

Code snippet showing usage

| | |
|---|---|
| **Type** | `string` |
| **Required** | Yes |
| **Min length** | 1 |

##### `description`

Explanation of what the example demonstrates

| | |
|---|---|
| **Type** | `string` |
| **Required** | Yes |
| **Min length** | 1 |

##### `title`

Short title for the example

| | |
|---|---|
| **Type** | `string` |
| **Required** | Yes |
| **Min length** | 1 |



### `featured`

Whether to feature this extension in UI/listings

| | |
|---|---|
| **Type** | `boolean` |
| **Required** | No |


### `license`

SPDX license identifier

| | |
|---|---|
| **Type** | `string` |
| **Required** | Yes |
| **Min length** | 1 |


### `longDescription`

Detailed multi-line description with markdown support

| | |
|---|---|
| **Type** | `string` |
| **Required** | Yes |
| **Min length** | 1 |


### `lua`

Lua extension configuration

| | |
|---|---|
| **Type** | `object` |
| **Required** | No |
| **Required when** | `type` is `lua` |

#### Properties

##### `inline`

Inline Lua script code

| | |
|---|---|
| **Type** | `string` |
| **Required** | No |
| **Min length** | 1 |

##### `path`

Path to the Lua script file

| | |
|---|---|
| **Type** | `string` |
| **Required** | No |
| **Min length** | 1 |


**Note:** One of the following must be specified: `path` or `inline`.


### `name`

Unique identifier for the extension

| | |
|---|---|
| **Type** | `string` |
| **Required** | Yes |
| **Min length** | 1 |
| **Pattern** | `^[a-z][a-z0-9-]*$` |


### `tags`

Searchable tags for categorization

| | |
|---|---|
| **Type** | `array of string` |
| **Required** | Yes |
| **Min items** | 1 |
| **Unique items** | Yes |
| **Item pattern** | `^[a-z][a-z0-9-]*$` |
| **Item min length** | 1 |


### `type`

Type of the extension

| | |
|---|---|
| **Type** | `string` |
| **Required** | Yes |
| **Allowed values** | `lua`, `wasm`, `dynamic_module`, `composer` |


### `version`

Semantic version of the extension

| | |
|---|---|
| **Type** | `string` |
| **Required** | Yes |
| **Pattern** | `^\d+\.\d+\.\d+$` |
