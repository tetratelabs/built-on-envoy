# <CLIName/> config

Manage CLI configuration.

## Usage

```bash
<CLIName/> config <command> [options]
```

## Commands

### `get <key>`
Get configuration value

```bash
<CLIName/> config get proxy.port
```

### `set <key> <value>`
Set configuration value

```bash
<CLIName/> config set envoy.version 1.37.0
```

### `list`
List all configuration

```bash
<CLIName/> config list
```

### `reset`
Reset to defaults

```bash
<CLIName/> config reset
```

## Configuration Keys

### `envoy.version`
Default Envoy version

### `proxy.port`
Default proxy port

### `admin.port`
Default admin port

### `log.level`
Default log level

### `marketplace.url`
Marketplace URL

### `editor`
Default editor for manifest files
