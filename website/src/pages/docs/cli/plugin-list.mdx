# <CLIName/> plugin list

List available extensions in the marketplace.

## Usage

```bash
<CLIName/> plugin list [options]
```

## Options

### `--category <name>`
Filter by category

Example: `--category security`

### `--tag <name>`
Filter by tag

Example: `--tag authentication`

### `--format <format>`
Output format (default: table)

Options: `table`, `json`, `yaml`

### `--featured`
Show only featured extensions

## Examples

### List all extensions
```bash
<CLIName/> plugin list
```

### List security extensions
```bash
<CLIName/> plugin list --category security
```

### List as JSON
```bash
<CLIName/> plugin list --format json
```

### List featured extensions
```bash
<CLIName/> plugin list --featured
```
