# Getting Started

The <code><CLIName/></code> (<ProjectName/>) CLI provides a zero-friction developer experience for discovering, running, and building Envoy® Proxy extensions.

## Installation

Choose your preferred installation method:

### Install Script

```bash
curl -sL https://get.built-on-envoy.io | bash
```

### Build from Source

```bash
git clone https://github.com/tetratelabs/built-on-envoy
cd built-on-envoy/cli
make
```

## Quick Start

Once installed, you can immediately start using extensions:

### List Available Extensions

<CodeBlock code="{{CLI_NAME}} list" />

### Run an Extension

<CodeBlock code="{{CLI_NAME}} run --extension rate-limiter" />

### Run Multiple Extensions

<CodeBlock code="{{CLI_NAME}} run --extension rate-limiter --extension cors" />

### Run a Local Extension

<CodeBlock code="{{CLI_NAME}} run --local ./my-extension" />

### Generate Envoy Configuration

<CodeBlock code="{{CLI_NAME}} gen-config --extension rate-limiter > envoy.yaml" />
