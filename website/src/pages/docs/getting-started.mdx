# Getting Started

The <code><CLIName/></code> (<ProjectName/>) CLI provides a zero-friction developer experience for discovering, running, and building Envoy® Proxy extensions.

## Installation

Choose your preferred installation method:

### Homebrew (macOS/Linux)

```bash
brew install tetratelabs/tap/built-on-envoy
```

### Install Script

```bash
curl -sL https://get.built-on-envoy.io | bash
```

### Build from Source

```bash
git clone https://github.com/tetratelabs/built-on-envoy
cd built-on-envoy
make install
```

## Quick Start

Once installed, you can immediately start using extensions:

### Run an Extension

```bash
# Run an extension from the marketplace
<CLIName/> run --plugin rate-limiter
```

### Create Your Own Extension

```bash
# Create a new extension
<CLIName/> create-plugin my-filter

# Test your extension locally
<CLIName/> run --plugin ./my-filter
```

### Browse Available Extensions

```bash
# List all available extensions
<CLIName/> plugin list

# Get details about a specific extension
<CLIName/> plugin info rate-limiter
```
